This website is built with modern, lightweight technologies:

- **React 19** — For dynamic, responsive user interfaces

## 🎯 Getting Started

### Prerequisites

- **Node.js** 16+ (check with `node --version`)
- **npm** 7+ (check with `npm --version`)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Guilhermedev25/ronei-and-naomi-tour.git
   cd ronei-and-naomi-tour
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The site will open at `http://localhost:5173/ronei-and-naomi-tour/`

### Available Commands

| Command           | Purpose                                        |
| ----------------- | ---------------------------------------------- |
| `npm run dev`     | Start local development server with hot reload |
| `npm run build`   | Build for production (outputs to `dist/`)      |
| `npm run preview` | Preview the production build locally           |
| `npm run lint`    | Check code quality with ESLint                 |

## 🌐 Deployment

This project is automatically deployed to **GitHub Pages** whenever you push to the `main` branch.

**Custom Domain:** [roneiandnaomitour.com](https://roneiandnaomitour.com)

### How It Works

- **GitHub Actions Workflow** (`/.github/workflows/deploy.yml`) automatically:
  1. Builds the site when you push to `main`
  2. Deploys to GitHub Pages
  3. Updates the live site instantly

No manual deployment needed! Just commit and push.
